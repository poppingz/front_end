selector

1. id
   고유해야 함.페이지에 딱 1번만! 중복사용불가(태그가 다르더라도)
   어떠한 대상 하나만 이용해서 작업을 할 때 사용
   back_end개발에서 많이 씀 front_end에서는 class 많이 씀

2. class
   중복사용가능
   공통 스타일 등 적용할 떄

3. tag

id > class > tag (우선순위)

  -ex) <div id="header" class="header"></div>

CSS 포함방법

1.inline 방법 (태그 옆에 attribute속성 입력)

  -ex) <div style="color:red"></div>

2.embed (=internal, head태그 영역에 <style></style>로 묶어서 CSS언어 입력)

  -ex)

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title></title>
    <!-- CSS -->
    <style>
      p{color:red}
    </style>
    <!--// CSS -->
</head>

3.external (링크로 파일을 불러옴,CSS구문을 HTML내에 작성하지 않고 파일을 불러옴)

 -ex) 파일 : A.CSS

      <!DOCTYPE html>
      <html lang="en">
      <head>
          <meta charset="UTF-8">
          <title></title>
          <!-- CSS -->
          <link rel="stylesheet" href="style.css"/>
          <!--// CSS -->
      </head>

문법(CSS)

selector(선택자) {
                 property(속성) : property 값;
                 property(속성) : property 값;
                 property(속성) : property 값;
                                             }


CSS에서 앞에 #이 붙어있으면 id 속성
            .이 붙어있으면 class 속성
           p.이 붙어있으면 p(해당태그)와 class
           #.이 붙어있으면 p(해당태그)와 id

가능한 조합 - p.center / p#center / #para1.center / .para1 center / #para1#center
           (tag+class)  (tag+id)   (id+클래스)        (class)        (id+id)

불가능한 조합 - pdiv (태그+태그)

조합할땐 우선순위대로 작성해야함

<p class="cente large"> => 이름이 center와 large 2개, 스페이스바로 구분, id도 동일
